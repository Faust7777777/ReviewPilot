# ReviewPilot

> AI PR Review 助手 — 帮助开发者提升 Pull Request 的 Review 效率与质量。

用户指定一个 GitHub PR，ReviewPilot 自动获取代码变更并做智能分析，辅助发现问题。核心能力：

- **PR 变更总结** — 把一组 diff 提炼成「这次改了什么、为什么改」。
- **风险代码识别** — 标出潜在 bug、边界遗漏、破坏性改动。
- **Review 建议生成** — 给出可操作的评审意见，而非泛泛而谈。

## 设计要点（待展开）

- **模型选择**：分析准确性、误报/漏报控制与响应速度之间的权衡。
- **上下文获取**：如何在有限上下文里喂给模型「足够但不过载」的代码信息。
- **未来扩展**：多语言、团队规范注入、CI 集成等方向。
